## 1、项目介绍

基于SSH的论坛管理系统1拥有两种角色

用户：分类查看板块帖子、发帖、引用帖子，登录注册等

管理员：可以上传帖子和评论


## 2、项目技术

后端框架：SSH（Struts+Spring+Hibernate）

前端技术jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven项目：否
- 硬件环境：Windows 或者 Mac OS
